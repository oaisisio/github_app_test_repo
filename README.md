# Welcome to the Repository
This is an initial commit by the GitHub App.

## Local Development Setup

1. Copy the example environment file and fill in your values:
   ```bash
   cp .env.example .env
   ```
2. Update `.env` with real credentials (e.g., Stripe test keys, database connection details).
3. Start the service:
   ```bash
   npm install
   npm run dev
   ```

See [`.env.example`](.env.example) for a full list of required environment variables.
